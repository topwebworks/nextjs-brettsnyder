# Brett Snyder Portfolio

Static-first portfolio and resume site for Brett Snyder.

## Start Here
- `docs/northstar.md`
- `docs/build-plan.md`
- `docs/handoff.md`
- `AGENTS.md` or `CLAUDE.md` for the repo workflow

## Common Commands
```bash
npm install
npm run dev
npm run generate-project-data
npm run generate-resume
npm run build
npm run lint
```

## Working Rules
- Content stays in Markdown files under `src/app/.../content/`.
- Reusable UI belongs in `src/components/`.
- Edit `public/resume-brett-snyder.html` and regenerate outputs instead of editing the PDF directly.
